## Optimism Monorepo Documentation

The `docs/` directory contains Optimism documentation closely tied to the implementation details of the monorepo (https://github.com/ethereum-optimism/optimism).

The directory layout is divided into the following sub-directories.

- [`postmortems/`](./postmortems/): Timestamped post-mortem documents.
- [`security-reviews`](./security-reviews/): Audit summaries and other security review documents.
- [`fault-proof-alpha`](./fault-proof-alpha): Information on the alpha version of the fault proof system.
- [`handbook/sepolia-l2-deployment.md`](./handbook/sepolia-l2-deployment.md): Sepolia L2 deployment runbook using Tokamak `Deploy.s.sol` and one-click script.
- [`handbook/sepolia-l2-runtime-quickstart.md`](./handbook/sepolia-l2-runtime-quickstart.md): Quickstart for running `op-node`, `op-batcher`, `op-proposer`, and `op-challenger` after Sepolia deployment.
- [`handbook/sepolia-oneclick-execution-guide.md`](./handbook/sepolia-oneclick-execution-guide.md): End-to-end execution guide for the one-click Sepolia deployment/runtime script.
